# rstf User Docs

`rstf` is a Go-first framework for server-rendered web apps with React islands, typed server data, and live queries.

These docs describe the framework as it exists in this repository today. The current workflow is intentionally narrow and still experimental.

## Current Status

- The supported CLI workflow today is `rstf dev`.
- Project bootstrapping is still manual.
- Production build and deployment commands are not documented here because they are not yet part of the stable user-facing workflow.
- APIs and conventions may still change.
